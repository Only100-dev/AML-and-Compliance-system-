### Key Design Decisions
1. **Lazy Loading**: All 102 jurisdiction prompt modules use dynamic `import()` — only the active jurisdiction is loaded
2. **Prompt Isolation**: UAE prompts CANNOT leak into Bahrain conversations (Principle B)
3. **Jurisdiction Resolution**: Both GCC alpha-2 (AE, SA) and legacy codes (CBUAE, DFSA, FSRA) are supported
4. **Base Context Injection**: Every prompt is prepended with shared regulatory principles (FATF, Risk-Based Approach, etc.)
5. **goAML Module**: For non-UAE countries, the `goAML` module correctly references the local SAR filing system (SAMA SAR, CBB SAR, etc.)
6. **SME Verification Tags**: Thresholds requiring verification are marked with `// Verify with SME` in prompt text
7. **Social Insurance Rates**: Each country's labor prompt includes specific employer/employee contribution rates
