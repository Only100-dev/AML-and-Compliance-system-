- `jurisdictionId` defaults to 'AE' when not provided
- Review phase prompt is jurisdiction-agnostic (references "the jurisdiction's regulatory framework" instead of UAE-specific articles)
- `RegulatoryThresholds` interface has 12 fields covering SAR deadlines, CTR, UBO, FIU, retention, etc.
