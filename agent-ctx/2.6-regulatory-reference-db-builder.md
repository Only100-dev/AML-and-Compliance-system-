## Key Design Decisions
- `NormalisableRef` interface handles both `module`-based (CBB/QCB/CBOA/CBK) and `ruleNumber`-based (SAMA) reference structures
- Dynamic imports in `queryRegulatoryReferences()` for code-splitting
- UAE (AE) returns empty from queryRegulatoryReferences (handled by existing REGULATORY_KNOWLEDGE_BASE)
- Backward-compatible: `jurisdiction` parameter defaults to 'AE' in tools.ts
- `// Verify with SME` comments for all unverifiable threshold values
